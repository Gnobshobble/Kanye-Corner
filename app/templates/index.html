Our Process

Learning Journey

We first started our journey by learning the basics of python and the various tools we would need to finish our Kanye AI. We did this via a number of notebooks where we could all interact and come up with fixes to various python coding problems. After learning python and creating our models, we needed to learn CSS and HTML to create this very website. Throughout this project we continued to learn various things and overall gained experience with coding and working with websites.

Brainstorming Ideas

Our group's brainstorm branched out to many ideas as we continue to develop our expertise with python. We originally came up with the idea of creating a code review bot, but then we realized it was beyond our capabilities. So, we decided to the next best thing which was creating a Kanye West Bot. This idea underwent much trial and error, but we finalized on creating two models in which one of them would create Kanye West-inspired lyrics and tweets, and the other would be a Kanye West chat bot.

Creating/Training the AI Model

To start, we created and trained the two different bots, one training off Kayne's tweets and one training off Kanye's lyrics. Since we are using two different models we had to train each individually. After training both, we realized we could upgrade our Kanye tweets bot from a GPT-2 to a GPT-3 model to improve its overall quality.

Challenges We Faced

Throughout this process, we have come across numerous challenges. The majority of these challenges come from training/creating/refining the models and their outputs. There were many challenges but an example is that our GPT-3 model was failing to properly respond to the prompt the user was asking. To fix this we made a change which had the model now find the most important word in the sentence and use that as the prompt. Besides the model, most of the issues came from creating our website. This includes things like a header not being centered on the page or some things just straight up not working. While numerous issues have come up throughout the project, in the end we were able to work everything out and complete our goal.
